
# Cash Hub
An invoice generator for foreign exchange

![Build](https://img.shields.io/wercker/ci/wercker/docs.svg)
![License](https://img.shields.io/npm/l/express.svg)

## Installation
```
git clone https://github.com/amalshehu/cash-hub.git
cd cash-hub
npm install
npm start
```
